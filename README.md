# ESP-GasSensor

*Display the level of gas with espNodMCU , using ITFFF.*

***********************************************************************************************************

                        Steps:

1- Electronic:
   * 1.1- prepare the circuit.
   * 1.2- prepare the code using # Arduino IDE-language C # .

2- IFTTT:
   * 2.1- ifttt it used to link between two different elements linked to the internet( like Iot).
   * 2.2- link the electronic circuit to its own sheet .  
   * 2.3- add the information from ifttt to the circuit’s code.

3- google sheet:
   * 3.1- make a table containing the title who created in ifttt ( I chose the titles).
   * 3.2- making sure that the path in google drive is the same as that in ifttt.
   * 3.3- check if the ifttt’s applets are true ( by clicking on check in the ifttt’s site).
   
4- web site:
   * 4.1- A coding website contains the sheet ( bring it from the sheet app).
        Using # vsCode- html/JS/CSS # .
            (optional):
              Make a login code using the Flask method in python >>> # vsCode- Flask - py #
